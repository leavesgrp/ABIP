\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00012}00012 \textcolor{comment}{ *}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00013}00013 \textcolor{comment}{ * The routines compute the LDL' factorization of a real sparse symmetric}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00014}00014 \textcolor{comment}{ * matrix A (or PAP' if a permutation P is supplied), and solve upper}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00015}00015 \textcolor{comment}{ * and lower triangular systems with the resulting L and D factors.  If A is}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00016}00016 \textcolor{comment}{ * positive definite then the factorization will be accurate.  A can be}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00017}00017 \textcolor{comment}{ * indefinite (with negative values on the diagonal D), but in this case no}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00018}00018 \textcolor{comment}{ * guarantee of accuracy is provided, since no numeric pivoting is performed.}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00019}00019 \textcolor{comment}{ *}}

\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00022}00022 \textcolor{comment}{ * indices in Ai [Ap [j] ... Ap [j+1]-\/1].  Ap [0] = 0 is required, and thus}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00023}00023 \textcolor{comment}{ * nz = Ap [n] is the number of nonzeros in A.  Ap is an int array of size n+1.}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00024}00024 \textcolor{comment}{ * The int array Ai and the abip\_float array Ax are of size nz.  This data structure}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00025}00025 \textcolor{comment}{ * is identical to the one used by MATLAB, except for the following}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00026}00026 \textcolor{comment}{ * generalizations.  The row indices in each column of A need not be in any}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00027}00027 \textcolor{comment}{ * particular order, although they must be in the range 0 to n-\/1.  Duplicate}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00028}00028 \textcolor{comment}{ * entries can be present; any duplicates are summed.  That is, if row index i}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00029}00029 \textcolor{comment}{ * appears twice in a column j, then the value of A (i,j) is the sum of the two}}

\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00032}00032 \textcolor{comment}{ * entries.}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00033}00033 \textcolor{comment}{ *}}

\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00043}00043 \textcolor{comment}{ *}}

\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00120}00120 \textcolor{comment}{ *}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00121}00121 \textcolor{comment}{ * LDL computes the symbolic factorization by finding the pattern of L one row}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00122}00122 \textcolor{comment}{ * at a time.  It does this based on the following theory.  Consider a sparse}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00123}00123 \textcolor{comment}{ * system Lx=b, where L, x, and b, are all sparse, and where L comes from a}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00124}00124 \textcolor{comment}{ * Cholesky (or LDL') factorization.  The elimination tree (etree) of L is}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00125}00125 \textcolor{comment}{ * defined as follows.  The parent of node j is the smallest k > j such that}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00126}00126 \textcolor{comment}{ * L (k,j) is nonzero.  Node j has no parent if column j of L is completely zero}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00127}00127 \textcolor{comment}{ * below the diagonal (j is a root of the etree in this case).  The nonzero}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00128}00128 \textcolor{comment}{ * pattern of x is the union of the paths from each node i to the root, for}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00129}00129 \textcolor{comment}{ * each nonzero b (i).  To compute the numerical solution to Lx=b, we can}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00130}00130 \textcolor{comment}{ * traverse the columns of L corresponding to nonzero values of x.  This}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00131}00131 \textcolor{comment}{ * traversal does not need to be done in the order 0 to n-\/1.  It can be done in}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00132}00132 \textcolor{comment}{ * any "{}topological"{} order, such that x (i) is computed before x (j) if i is a}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00133}00133 \textcolor{comment}{ * descendant of j in the elimination tree.}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00134}00134 \textcolor{comment}{ *}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00135}00135 \textcolor{comment}{ * The row-\/form of the LDL' factorization is shown in the MATLAB function}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00136}00136 \textcolor{comment}{ * ldlrow.m in this LDL package.  Note that row k of L is found via a sparse}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00137}00137 \textcolor{comment}{ * triangular solve of L (1:k-\/1, 1:k-\/1) \(\backslash\) A (1:k-\/1, k), to use 1-\/based MATLAB}}
\DoxyCodeLine{\Hypertarget{ldl_8c_source_l00138}00138 \textcolor{comment}{ * notation.  Thus, we can start with the nonzero pattern of the kth column of}}
